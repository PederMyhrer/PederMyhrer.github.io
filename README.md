# 💸 Budsjett App

Dette er en enkel budsjettapp for å få bedre oversikt over økonomien din 📊  

## Funksjoner
- 📅 Følge med på inntekter og utgifter  
- 🧾 Se hvor pengene dine går  
- 🎯 Sette mål for sparing  
- 📈 Få en rask oversikt over økonomisk status  

Hold kontroll, spar smartere, og ta bedre økonomiske valg 💡
